# ClassScoreManageSystem-koishi-plugin
让CSMS与QQ通联，基于koishi的插件开发
直接在koishi管理页面-插件市场-搜索【class-score-system】安装即可，插件会不定时更新，有问题或参与开发请回到[CSMS主项目](https://github.com/QianKunBoss/ClassScoreManageSystem "CSMS")页面加群（请安装后自行手动重启koishi系统）
